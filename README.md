🧩 Number Puzzle Game

 About the Project:

Hello! My name is Vineela, and I developed this Number Puzzle Game using Python and Tkinter.

This project is a simple puzzle game where the player has to arrange the shuffled numbers in the correct order. While building this project, I improved my understanding of Python programming, GUI development, event handling, and game logic.



 Features

.Interactive 3x3 Number Puzzle

. Move Counter

. Timer System

. Shuffle Button

. Restart Button

. Winning Message

. User-Friendly Interface


Technologies Used

- Python
- Tkinter

 How to Run the Project

1. Install Python on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:


python puzzle.py

How to Play

1. The puzzle starts with shuffled numbers.
2. Click a tile next to the empty space to move it.
3. Arrange the numbers from **1 to 8** in the correct order.
4. Complete the puzzle using the minimum number of moves and time.
